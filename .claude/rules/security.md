# **Security Rules**

## **通信**

* API Gateway には適切な CORS 設定を行い、特定のドメイン（または開発環境）のみ許可すること。

## **データ保護**

* **S3 Access:** バケットはパブリック公開せず、CloudFront OAC または IAM Role を通じてのみアクセスを許可する設計とする。  
* **Validation:** \- アップロードされるファイルが .md 拡張子であることをサーバー側で検証すること。  
  * ファイルサイズの上限（例: 5MB）を設け、過大なデータ送信を拒否すること。

## **入力チェック**

* タイトルやカテゴリなどの自由入力項目に対して、サニタイズ処理を行うこと。
